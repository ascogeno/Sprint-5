# Trapped Box picture changing app, with occasional sound

The player is trapped in a box, where they can move around in a limited space and change the picture on the wall in front of them. There are supposed to be videos that play, but only the audio works

## Instructions for Build and Use

Steps to build and/or run the software:
TWO WAYS TO DO THIS:
Clone Repo:
1. Clone the Unity repository
2. Download and install Unity
3. Once Unity is open, import the cloned files and open the project
4. Press the play button in the top middle of the screen

1. Take android phone, connect to computer via USB (or some other file sharing solution. That's what we need this for)
2. Take APK from the secondary repo and copy to phone
3. Follow instructions from the phone for installing the app (I installed it on mine, I swear its safe. Look through the former repo if you want to double check)
4. Open the app
5. (I suppose you could also install the android build tools in unity and build your own APK. If that's your course of action, follow this set of steps from step 1 after building the APK)

Instructions for using the software:

1. Move the player using the "joystick" in the bottom left corner
2. Tap the "Tap" text to change the picture on the wall
3. Close the app
